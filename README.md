# Prime Numbers in Java

This Java program prints all prime numbers up to a given number entered by the user.

## Features
- Takes user input
- Checks for prime numbers efficiently
- Uses a separate method for prime checking
- Displays all prime numbers up to the given number

## Technologies Used
- Java
- Scanner class
- Functions (methods)

## How to Run

1. Compile:
   javac PrimeNumbers.java

2. Run:
   java PrimeNumbers

3. Enter a number to see all prime numbers up to that value.

## Example

Input:
10

Output:
2 3 5 7
